# Bong's Portfolio

## 1. 프로젝트 소개

- Next.js App Router 기반 포트폴리오 웹사이트입니다.
- 실무에서 사용하는 인증 흐름(NextAuth v5) 을 데모 형태로 포함하고 있습니다.

- 포트폴리오는 로그인 없이 열람 가능
- 로그인 시 세션 상태 및 보호된 UI 흐름 체험 가능
- 실제 서비스에서 사용하는 인증 구조를 단순화하여 구현

## 2. 주요 기능

- 공개 포트폴리오 페이지
- Credentials / GitHub OAuth 로그인
- JWT 기반 세션 관리
- Middleware 보호 라우트
- 인증 상태에 따른 UI 뱃지 및 사용자 정보 표시

## 3. 인증 흐름 (Auth Flow)

- Client
  ↓ 로그인 요청
- NextAuth (Credentials / OAuth)
  ↓ JWT 발급
- Session 유지
  ↓
- Middleware
  ├─ 인증됨 → 접근 허용
  └─ 미인증 → /login 리다이렉트

### Middleware를 사용이유

- 클라이언트 보호만 사용할 경우 초기 렌더링 깜빡임 발생
- 서버 레벨에서 라우트 접근을 제어
- 불필요한 렌더링 방지
- 실무 환경과 유사한 인증 구조 학습 목적

### 설계 포인트

- `/portfolio`는 공개 접근 허용
- 인증 여부에 따라 UI 상태 배지 변경
- 인증은 “필수 기능”이 아닌 UX 체험 요소로 설계

### 설계 이유

- 포트폴리오는 로그인 없이 바로 확인 가능해야 함
- 인증 구조는 "인증 흐름"을 보여주는 용도
- UX 관점에서 인증을 자연스럽게 표시
